Overview

The DigitalEnergy MCP Gateway provides a unified interface for AI assistants and agents to interact with Oil & Gas enterprise systems. Built on the Model Context Protocol (MCP), it enables seamless integration between LLMs and industrial systems including SCADA, historians, ERP, and production accounting platforms.

Quick Start

Install via Docker

docker pull digitalenergy/mcp-gateway:latest
docker run -d -p 8080:8080 digitalenergy/mcp-gateway

Or using Docker Compose

curl -O https://digitalenergyca.github.io/docker-compose.yml
docker-compose up -d

Authentication Example

# JWT Authentication
curl -H "Authorization: Bearer eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9..." \
     https://gateway.digitalenergy.ca/mcp/tools

Configuration Examples

REST Endpoint to MCP Tool

{
  "name": "get_well_production",
  "description": "Retrieve daily production data for a specific well",
  "inputSchema": {
    "type": "object",
    "properties": {
      "well_id": {"type": "string"},
      "date_range": {"type": "string", "format": "date"}
    }
  },
  "endpoint": {
    "url": "https://api.company.com/wells/{well_id}/production",
    "method": "GET",
    "auth": "bearer_token"
  }
}

Federation Example

federation:
  namespaces:
    - name: "field_ops"
      servers:
        - "scada-mcp-server"
        - "historian-mcp-server"
    - name: "corporate"
      servers:
        - "sap-mcp-server"
        - "maintenance-mcp-server"
  discovery:
    auto_register: true
    health_check_interval: 30s
